### タイトル（Issue タイトルに入力してください）

import 

### どんな技術でどういう仕組み？

- tsconfigで「`@` はこのディレクトリを指しているよ」と教えてあげる必要がある
- ただ、baseUrlは、TypeScript 7.0で非推奨」となったみたい。

```
{
  "compilerOptions": {
    "baseUrl": ".", // プロジェクトのルートディレクトリを指定
    "paths": {
      "@/*": ["src/*"] // "@/" が "src/" を指すように設定
    }
  }
}
```

### 何を解決できる？

長いパスを省略して、使えるようになった。

従来
- import { postBooking } from "../api/post-booking";

使うと
- import { postBooking } from "@/api/post-booking";
